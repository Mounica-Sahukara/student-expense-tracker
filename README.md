# 💰 Student Expense Tracker

A modern and responsive **Expense Tracker Web Application** built using **HTML, CSS, and JavaScript**.

This application helps students manage daily expenses by selecting categories and tracking transactions in a clean and interactive UI.

---

## 🚀 Features

* 📂 20 predefined expense categories
* ➕ Add transactions easily
* 🧹 Clear all transactions
* 🎨 Modern UI with gradient background
* ✨ Hover effects on cards
* 📱 Fully responsive design
* ⚡ Smooth and interactive user experience

---

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript

---

## 📸 How It Works

1. Select a category (Food, Shopping, Bills, etc.)
2. Enter the amount
3. Click **Add Transaction**
4. View all transactions in the list
5. Use **Clear All** to reset
